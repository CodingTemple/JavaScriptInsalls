Mac JavaScript Installs

Step 1 of 1 Install NodeJS, iJavaScript and TSLab

Steps in Video

  • brew install pkg-config node zeromq
  • node -v
  • npm -v
  • pip install --upgrade pyzmq jupyter You may need to replace pip with pip3
  • npm install -g ijavascript
  • ijsinstall
  • jupyter kernelspec list
  • If you receive the error conda not found
    • Find your path to your anaconda3 folder and use to replace /path/to/ part below
    • /path/to/anaconda3/bin/conda init zsh
    • conda activate
    • conda deactivate
  • npm install -g tslab if that fails run brew install cmake and re-run inital command
  • tslab install --version
  • tslab install
  • conda activate
  • jupyter notebook
It is possible you get an error at some point talking about xcode
If that is the case then run the following command and rerun the command that gave you the error
xcode-select --install
